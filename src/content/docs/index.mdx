## What is sverm.dev?

This is the **machine room**. [sverm.ai](https://sverm.ai) is the product — this is
where you learn how it works, how to build with it, and how to extend it.

We don't do marketing here. We do documentation, benchmarks, and the occasional
honest blog post about what we've learned.

:::note
This site and the sverm platform are **in active development**.
The documentation describes the target architecture and API — some
features are still being built. For questions or early access:
[raymond@sverm.ai](mailto:raymond@sverm.ai).
:::
