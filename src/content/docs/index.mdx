## What is sverm.dev?

**[sverm.ai](https://sverm.ai)** is the product. Organizations use it to run
structured debates between AI personas, improving decisions through
constructive dissent.

**sverm.dev** is the machine room. Everything you need to understand,
build, and extend the platform yourself. We document how agents work,
how cost routing decides which model to use, and how to deploy swarms
at scale.

Same platform. Two doors.

We don't do marketing here. We do documentation, benchmarks, and the occasional
honest blog post about what we've learned.

:::note
This site and the sverm platform are **in active development**.
The documentation describes the target architecture and API — some
features are still being built. For questions or early access:
[raymond@sverm.ai](mailto:raymond@sverm.ai).
:::
