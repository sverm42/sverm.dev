The playground is an interactive sandbox where you can run AI swarms
directly in your browser. No registration. No API keys. Just curiosity.

---

## How it works

1. Pick a swarm template
2. Type your task
3. Watch the agents work — live

```
┌──────────────────────────────────────────────┐
│  Try a swarm — no registration needed         │
│                                              │
│  Template:  [Research ▼]                     │
│  Task:      [___________________________]    │
│                                              │
│  [▶ RUN SWARM]                               │
│                                              │
│  ──────────────────────────────────────      │
│  Status: •••○○  (3/5 agents complete)        │
│  Cost so far: $0.04                          │
│                                              │
│  researcher-1  Searching for: ...            │
│  researcher-2  Found 4 sources: ...          │
│  manager       Synthesizing results...       │
└──────────────────────────────────────────────┘
```

## Current status

:::caution
The playground is being built. It will be available at
[https://sverm.ai/playground](https://sverm.ai/playground) with
the following features:
:::

- **4 pre-built swarm templates** (research, code review, debate, writing)
- **Live agent feed** — watch agents think, search, and communicate
- **Real-time cost counter** — always know what a swarm costs
- **Open source** — the playground itself will be on GitHub

### Limits (to keep it free)

- 3 swarms per hour per IP address
- Max $0.50 cost per swarm
- Only pre-built templates (no custom YAML)

These limits let us offer the playground for free indefinitely while
protecting against abuse.

---

## Why a playground?

Because reading about swarms isn't the same as watching one work. The
playground is designed to create an "aha!" moment — when you see 5
agents coordinating in real time, the concept clicks in a way that
documentation alone can't achieve.

We fund the playground's token costs ourselves. It's our way of saying:
"This is real. Try it yourself."
